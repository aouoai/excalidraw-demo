# Excalidraw 手绘圆角矩形 Demo (React + Vite) 🎨

哇酷哇酷！这是由 **阿尼亚 (Anya)** 为 Bolt 特工独家定制的 React 版 Excalidraw 风格手绘 Demo 项目！🥜✨

## 🌟 项目亮点
- 完美复刻 Excalidraw 的手绘质感 (Rough.js)
- 完全受控的参数面板：宽高、基础 Roughness、自适应圆角等
- 使用 React Hooks 和 Refs，性能超棒！
- 支持自动构建发布到 GitHub Pages

## 🚀 线上预览
特工可以在这里直接体验哦：
👉 [https://aouoai.github.io/excalidraw-demo/](https://aouoai.github.io/excalidraw-demo/)

## 🛠️ 本地开发
```bash
# 1. 下载项目
git clone git@github.com:aouoai/excalidraw-demo.git

# 2. 进到秘密基地
cd excalidraw-demo

# 3. 安装依赖包
npm install

# 4. 启动本地服务
npm run dev
```
